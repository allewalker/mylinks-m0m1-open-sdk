//
//	安装M0M1系列编译器
//
///////////////////////////////////////////////////////////////////////////////
1. 从百度云盘下载编译器
链接:http://pan.baidu.com/s/1nuW643n  密码:ljo7

2.与mylinks-m0m1-open-sdk同级目录下新建文件夹:toolchain
3.将ba-elf_4.7.3.tgz压缩到toolchain中


///////////////////////////////////////////////////////////////////////////////
//
//	编译项目并生成烧写文件img
//
///////////////////////////////////////////////////////////////////////////////
1. 在 mylinks-m0m1-open-sdk/proj 中有at文件夹，即为一个at项目.
	在 mylinks-m0m1-open-sdk 路径下使用以下命令
	"make clean-at"	-> 清除 proj/at
	"make at"			-> 编译 at.img
2. 当编译完成后, at.img 会出现在 mylinks-m0m1-open-sdk/images 中.


注：编译环境为linux。
